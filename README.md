# Odin Recipes

My first project from [The Odin Project](https://www.theodinproject.com/):
a recipe website built with plain HTML.

The site consists of an index page that links to several recipe pages.
The goal is to practice core HTML skills; styling with CSS will be added
later in the curriculum.

## Skills demonstrated

- HTML page structure (boilerplate, head, body)
- Headings, paragraphs and lists
- Relative links between pages of the same site
- Images with `src`, `alt`, `width` and `height` attributes
- Git workflow: clone, add, commit, push

## Project status

In progress — HTML stage of The Odin Project curriculum.
